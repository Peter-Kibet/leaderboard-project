## ğŸ’» Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps downbelow!

### Prerequisites

In order to run this project you need:

- Web browser
- Git-syn
- Code editor
- Node.js
- npm (Node Package Manager)

### Setup

1. Clone this repository to your desired folder:

git clone [leaderboards](https://github.com/Peter-Kibet/leaderboard-project)

2. Navigate to the project's root directory:

cd Awesome-book

3. Install the dependencies:

npm install

### Install

Install this project run the command:

```

npm install

```

### Usage

To run the project in development mode, use the following command:

npm start

This will start the webpack dev server and launch the project in your default browser.

Build
To build the project for production, use the following command:

npm run build

This will create a optimized and minified version of the project in the dist directory.

### Run tests

To run tests, run the following command:

- Track HTML linter errors run:

```
npx hint .
```

- Track CSS linter errors run:

```
npx stylelint "**/*.{css,scss}"
```

- Track JavaScript linter errors run:

```
npx eslint .
```
